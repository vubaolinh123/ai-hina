# Asset manifests

The repository-original `hina.code-avatar.v1` SVG/CSS fallback is approved for
local development and redistribution under the repository MIT license.
`vrm1.constraint-twist-sample.v1` is the fixed local VRM development base with
embedded VRM Public License metadata, and `hina.kawaii-vrm-presentation.v1`
records the repository-original palette, pose and accessory layer applied to
that unchanged base. Neither manifest claims a final/exclusive Hina identity.
The bundled voice preset remains local-evaluation only until independent
speaker-consent evidence exists.

Each future manifest must record owner/source, license, permitted use, attribution, consent, deletion rights and hashes. Publicly reachable audio is not consent for voice cloning.
