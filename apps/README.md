# Applications

Các ứng dụng có thể chạy của Hina:

- `core-runtime`: Python runtime sở hữu control plane, realtime transport,
  persistence, lifecycle và observability.
- `dev-console`: ứng dụng web local để owner thao tác trực tiếp với runtime M01.
- `desktop`: Electron/Vue stage và operator UI, dự kiến bắt đầu ở M07.

Mỗi ứng dụng phải có manifest và test riêng trước khi thêm mã.
